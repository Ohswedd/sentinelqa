# docs/

Project documentation. PRD §11.2, CLAUDE §34.

- `adr/` — Architecture Decision Records. Mandatory for every trigger in `CLAUDE.md` §34.
- `dev/` — Contributor docs: local setup, branching, commits, secret hygiene, ownership, CI, status labels, agent workflow.
- `user/` — End-user docs (CLI usage, SDK reference, module catalog; Phase 27).

Every doc carries a status label from `CLAUDE.md` §34 (`Planned`, `Experimental`, `Stable`, `Deprecated`). See `docs/dev/status-labels.md` (Phase 00.09).
